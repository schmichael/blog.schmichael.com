How to Build a Web2.0 Link Sharing Site

  2007-09-05

While I already posted about the excellent Genshi Tutorial, the author, cmlenz, has continued adding to it. The tutorial is now an excellent introduction to building a reddit/digg style link sharing site complete with pretty AJAX (using jQuery), Atom Feeds, and [HTML posting with XSS sanitizing][8].

The only two things missing are user accounts and voting. Both of which are far beyond the scope of a templating engine tutorial!

[8]: http://genshi.edgewall.org/wiki/GenshiTutorial#AllowingMarkupinComments
