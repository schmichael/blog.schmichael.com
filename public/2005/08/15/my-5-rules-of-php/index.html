My 5 Rules of PHP

  2005-08-15

  1. Never use register_globals.

  2. Never use magic_quotes_*.

  3. Never use eval().

  4. Never use extract() – aka Quit being so lazy.

  5. Never put $_REQUEST variables into SQL queries unescaped.
