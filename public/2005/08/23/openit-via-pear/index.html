OpenIT via PEAR?

  2005-08-23

I didn’t realize [PEAR was going to become a means of distribution for PHP web applications.][1] I always thought of PEAR as being an easy way to get PHP extensions, but not whole applications. It probably would have helped if I had realized PEAR stands for PHP Extension and Application Repository.

I probably won’t attempt this with OpenIT until after PEAR 1.4 is released, but I have high hopes that this will make installation much easier.

[1]: http://www.killersoft.com/randomstrings/2005/08/22/5-Release-Your-Next-Project-as-a-PEAR-1.4.0-Package.html
