Boss Blogging About Venezuela

  2006-03-28

I haven’t even gotten around to blogging about Synthesys’s planned trip to Venezuela, but my boss, Bob Rinella, has been posting like crazy!

The story goes, Bob visited a school/church in Venezuela a few months ago and has been planning a work trip Synthesys there ever since. The plan is to set the school up with Edubuntu on old laptops for the children to learn basic typing and computing skills. With the Venezuelan government’s plan to move entirely to open source, learning Edubuntu will hopefully give these kids a boost in the world.

This is the sort of thing I’d love to do with the rest of my life: use open source software to help those who otherwise couldn’t afford computers or expensive proprietary software.

I still need to do lots of planning, so I’ll try to post more details as I figure them out.

Please don’t hesitate to [contact me][5] if you have any questions, comments, or contributions (laptops, finanacial support, advice, etc).
