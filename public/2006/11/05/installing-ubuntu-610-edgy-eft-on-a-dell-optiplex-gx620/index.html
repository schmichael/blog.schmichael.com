Installing Ubuntu 6.10 Edgy Eft on a Dell Optiplex GX620

  2006-11-05

I wanted to install Ubuntu on my work computer, a Dell Optiplex GX620, over the weekend. It turned out to be harder than I was hoping.

The bad news is that Ubuntu’s installer tries to use the ati video driver instead of the radeon video driver. You can hop into a console and fix it, but even users with a moderate level of Linux experience won’t feel comfortable doing that.

Once I got that sorted out it works beautifully. I’m writing this post in Ubuntu 6.10 using Firefox 2.0! Time to install some eye candy…
