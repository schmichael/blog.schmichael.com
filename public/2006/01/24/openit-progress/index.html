OpenIT Progress

  2006-01-25

I’ve been keeping up a good pace of bug fixing in OpenIT thanks to user bug reports. I’m really getting bitter about using MySQL 4.0, especially after we’ve started using PostgreSQL for a new project at Synthesys.

Views? Foreign Key Constraints? [Cascading UPDATEs and DELETEs][6]? I’ve had to implement these in PHP! Lord have mercy on my soul. While I appreciate how easy MySQL is to install and use, after using PostgreSQL for just a short while I’m ready to drop MySQL and never look back.

[6]: http://www.postgresql.org/docs/8.1/interactive/ddl-constraints.html#DDL-CONSTRAINTS-FK
