OpenSSH 4.3 Released

  2006-02-02

The release notes for the new version of [OpenSSH][2] are far too modest:

This is primarily a bug-fix release, only one new feature has been added:
* Add support for tunneling arbitrary network packets over a connection...

Basically this means OpenSSH is now a full VPN solution. While its only supported in Linux and some BSDs right now, I’ll be excited to see where this technology goes.

[2]: http://www.openssh.org/
