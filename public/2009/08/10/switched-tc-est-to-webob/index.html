Switched tc-rest to webob

  2009-08-11

Small update on my toy tc-rest project: I switched to using WebOb for creating HTTP Request and Response objects. Cleaned up the code a bit, but a real dispatcher is what’s needed to really remove the cruft.

I’m anxious to extend the API and add features, but I have no clue when I’ll have time to touch it again. In the mean time I’ve [pushed tc-rest to bitbucket.org if you want to take a look][2].

[2]: http://bitbucket.org/schmichael/tc-rest/overview/
